# Contrato de producción

Una ejecución está terminada únicamente cuando contiene `spec.json`, los PNG finales generados con GPT Image 2.0, una hoja de contacto final, copys separados para Instagram y LinkedIn, `sources.json`, `humanizer.json`, `imagegen.json`, `manifest.json`, `qa.json` y `qa.md`.

`APPROVED` exige formato 1080 x 1350, secuencia válida, fidelidad al estilo de `recursos/carrusel/`, assets completos, copys con la extensión mínima y política de honestidad aprobada.

El modo de publicación es `draft`. Aprobar el artefacto no autoriza publicar en una red social.
