# Contrato de producción

Una ejecución está terminada únicamente cuando contiene `spec.json`, siete SVG, siete PNG, `contact-sheet.svg`, copys, `sources.json`, `manifest.json`, `qa.json` y `qa.md`.

`APPROVED` exige formato 1080×1350, secuencia válida, acento #FF4B0B, densidad admisible, assets completos y política de honestidad aprobada.

El modo de publicación es `draft`. Aprobar el artefacto no autoriza publicar en una red social.
