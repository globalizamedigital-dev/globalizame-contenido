# Lead magnet

Comando manual. No crear ni depender de tareas programadas.

Usar `content-loop` (skill). Seleccionar la pieza pendiente que tenga CTA de recurso, construir el lead magnet desde el post y la evidencia de `recursos/`, pasar todo texto por la skill `humanizer` real y aplicar el control de honestidad. El CTA solo se activa cuando el archivo existe y ha superado QA.
