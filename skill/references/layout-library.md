# Biblioteca de layouts

| ID | Uso | Estructura |
|---|---|---|
| `cover-split` | Portada | Titular 7 columnas; metáfora 5 columnas; gran vacío central |
| `scene-left` | Contexto | Visual 5 columnas izquierda; texto 6 columnas derecha |
| `metric-stage` | Dato | Cifra ocupa 60% del lienzo; contexto en banda inferior |
| `matrix-100` | Proporción | Matriz 10×10 y conclusión directa |
| `causal-flow` | Consecuencia | Origen → pérdida → destino, lectura diagonal |
| `range-leak` | Síntesis | Escala vertical y objeto metafórico |
| `clock-ring` | Tiempo | Diagrama circular proporcional, etiqueta directa |
| `closing-loop` | Cierre | Titular dominante y motivo de portada reducido |

Regla: no usar el mismo ID dos veces en una pieza. Una excepción exige una justificación narrativa escrita en la Biblia visual.
